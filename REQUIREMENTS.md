# 한국 관광 정보 지도 앱 - 요구사항 명세서

> 작성일: 2026-02-11

---

## 1. 프로젝트 개요

### 1.1 목적
지도 기반으로 한국의 관광 정보(관광지, 맛집, 숙박, 쇼핑 등)를 직관적으로 탐색할 수 있는 반응형 웹 애플리케이션

### 1.2 핵심 가치
- **직관적 탐색**: 지도 위에서 한눈에 관광 정보를 파악
- **다국어 접근성**: 내/외국인 누구나 언어 장벽 없이 사용
- **모바일 우선**: 여행 중 모바일에서 즉시 활용 가능
- **확장 가능**: 한국 → 아시아로 데이터 확장 가능한 구조

## 4. 기능 요구사항

### 4.1 MVP (1차 릴리스) - 한국 지도 + 기본 POI

#### 4.1.1 지도 기능
- [ ] Mapbox GL JS 기반 한국 전체 지도 표시
- [ ] 카테고리별 마커/핀 표시 (관광지, 맛집, 숙박, 쇼핑 등)
- [ ] 마커 클러스터링 (줌 레벨에 따른 자동 그룹화)
- [ ] 마커 클릭 시 팝업/사이드 패널에 상세 정보 표시
- [ ] 줌 인/아웃, 드래그 이동
- [ ] 현재 위치 표시 (Geolocation API)

#### 4.1.2 검색 기능
- [ ] 키워드 검색 (장소명, 지역명)
- [ ] 검색 결과 지도 위에 하이라이트
- [ ] 검색 자동완성 (선택적)

#### 4.1.3 필터 기능
- [ ] 카테고리별 필터 (관광지/맛집/숙박/쇼핑/축제 등)
- [ ] 지역별 필터 (시/도 단위)
- [ ] 복수 필터 조합 지원

#### 4.1.4 상세 정보
- [ ] 장소 이름 (한/영)
- [ ] 카테고리 아이콘/라벨
- [ ] 주소
- [ ] 연락처 (있는 경우)
- [ ] 간략한 설명
- [ ] 이미지 (있는 경우)
- [ ] 외부 링크 (공식 사이트, 지도 앱 연결 등)

#### 4.1.5 다국어 (1차)
- [ ] 한국어 / 영어 2개 언어 지원
- [ ] 언어 전환 UI (헤더)
- [ ] URL 기반 로케일 (`/ko/...`, `/en/...`)

#### 4.1.6 반응형 레이아웃
- [ ] **PC (1024px+)**: 전체화면 지도 + 좌측 사이드 패널
- [ ] **Tablet (768px~1023px)**: 지도 + 하단 슬라이드업 패널
- [ ] **Mobile (< 768px)**: 지도 + 하단 바텀시트 (드래그 가능)

### 4.2 2차 확장 (Post-MVP)

- [ ] 추가 언어 지원 (중국어, 일본어 등)
- [ ] 아시아 지역 데이터 확장

## 5. 비기능 요구사항

### 5.1 성능
- 초기 로딩: LCP 2.5초 이내
- 지도 인터랙션: 60fps 목표
- 대량 마커 처리: 1,000개+ POI 클러스터링으로 성능 유지

### 5.2 SEO
- 각 관광지별 개별 상세 페이지 생성 (SSG)
  - URL 패턴: `/ko/spots/[slug]`, `/en/spots/[slug]`
- 메타태그, OG태그, 구조화된 데이터(JSON-LD) 적용
- sitemap.xml 자동 생성
- 메인 지도 페이지 + 개별 상세 페이지 이중 구조

### 5.3 접근성
- 키보드 네비게이션 지원
- 스크린리더 기본 대응 (ARIA 레이블)
- 충분한 색상 대비

### 5.4 브라우저 지원
- Chrome, Safari, Firefox, Edge (최신 2개 버전)
- iOS Safari, Android Chrome

## 6. 데이터 설계 (초안)

### 6.1 데이터 소스
- **원본**: 한국관광공사 데이터 (TourAPI 또는 공공데이터 다운로드)
- **가공**: 필요한 필드만 추출, 정제하여 JSON/GeoJSON으로 변환
- **저장**: 프로젝트 내 `/data` 디렉토리에 정적 파일로 관리

### 6.2 POI 데이터 스키마 (예상)

```typescript
interface POI {
  id: string;
  slug: string;                  // URL용 슬러그
  category: Category;            // 카테고리 enum
  coordinates: {
    lat: number;
    lng: number;
  };
  name: {
    ko: string;
    en: string;
  };
  address: {
    ko: string;
    en: string;
  };
  description?: {
    ko: string;
    en: string;
  };
  region: string;                // 시/도 코드
  images?: string[];             // 이미지 URL 배열
  contact?: string;              // 연락처
  website?: string;              // 공식 웹사이트
  tags?: string[];               // 검색/필터용 태그
  updatedAt: string;             // 데이터 갱신일
}

type Category =
  | 'attraction'    // 관광지
  | 'restaurant'    // 맛집
  | 'accommodation' // 숙박
  | 'shopping'      // 쇼핑
  | 'festival'      // 축제/행사
  | 'culture'       // 문화시설
  | 'nature'        // 자연
  | 'leisure';      // 레저/체험
```

### 6.3 GeoJSON 구조

```json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "geometry": {
        "type": "Point",
        "coordinates": [126.9780, 37.5665]
      },
      "properties": {
        "id": "seoul-tower-001",
        "slug": "namsan-seoul-tower",
        "category": "attraction",
        "name_ko": "남산서울타워",
        "name_en": "Namsan Seoul Tower",
        "region": "seoul"
      }
    }
  ]
}
```

## 7. 페이지 구조

```
/                         → 메인 지도 페이지 (전체화면 지도 + 사이드패널)
/ko                       → 한국어 메인
/en                       → 영어 메인
/ko/spots/[slug]          → 관광지 상세 페이지 (SEO용, SSG)
/en/spots/[slug]          → 영어 상세 페이지
```
